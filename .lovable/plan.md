

# خطة شاملة لتطوير عرض العملاء في قائمة العملاء

## ملخص الإجابات

| البند | الاختيار |
|---|---|
| حجم البطاقة | قابلة للتوسيع (مضغوطة افتراضياً) |
| المحتوى | اسم، رصيد، هاتف، موقع + إجراءات (عرض، فاتورة، دفعة) |
| الإجراءات | ضغط مطول (Long Press) |
| Avatar | مربع مدور |
| التخطيط | هجين (أفقي + عمودي) |
| التوسيع | KPIs مالية + آخر معاملات |
| الخلفية | تدرج خفيف + حافة ملونة حسب VIP |
| شريط الإحصاءات | إجمالي + chips فلترة سريعة |
| الفلاتر | chips سريعة + فلترة متقدمة |
| الترتيب | اسم، رصيد، تاريخ إضافة، آخر نشاط |
| العرض | قائمة + شبكة مع تبديل |
| البحث | يختفي عند التمرير |
| التحديد المتعدد | ضغط مطول |
| VIP | شارة نصية + إطار Avatar ملون |
| حالة النشاط | نقطة لونية + تباهت البطاقة |
| الترقيم | Infinite scroll |
| الحالة الفارغة | دليل خطوات |
| Grid موبايل | أعمدة تلقائية |
| ألوان | هجين (VIP + نوع + صحة مالية) |
| أنيميشن | توسيع سلس + fade-in + تأثير لمس |

## التفاصيل التقنية

### 1. مكون `CustomerListCard.tsx` (جديد)
بطاقة موبايل هجينة قابلة للتوسيع:

```text
الحالة المضغوطة (افتراضي):
┌─ حافة VIP ملونة ──────────────────────┐
│ [Avatar مربع]  اسم العميل    ● نشط   │
│  +إطار VIP     شركة · [VIP ذهبي]      │
│               📍 القاهرة  📞 01x       │
│ ─────────────────────────────────────  │
│  💰 15,000 ج.م        ▓▓▓▓░░ 75%      │
└────────────────────────────────────────┘

عند التوسيع (ضغط على البطاقة):
┌─────────────────────────────────────────┐
│  ... المحتوى المضغوط أعلاه ...          │
│ ═══════════════════════════════════════ │
│  KPIs: مستحق | مشتريات | نسبة السداد   │
│ ─────────────────────────────────────── │
│  آخر فاتورة: #123 — 5,000 ج.م — 2/3   │
│  آخر دفعة: 3,000 ج.م — 1/3             │
│ ─────────────────────────────────────── │
│  [عرض] [فاتورة جديدة] [دفعة جديدة]    │
└─────────────────────────────────────────┘
```

- خلفية: `bg-gradient-to-b from-primary/5` + `border-s-4` بلون VIP
- أنيميشن توسيع: `max-h` transition سلس + `animate-fade-in`
- تأثير لمس: `active:scale-[0.98]`
- Long press: يفتح `LongPressMenu` مع خيارات (عرض، تعديل، حذف، فاتورة، دفعة)
- غير النشط: `opacity-60` على البطاقة كاملة + نقطة رمادية
- Avatar مربع مدور مع إطار ملون حسب VIP (ذهبي، فضي، بلاتيني)

### 2. تعديل `CustomerAvatar.tsx`
- إضافة prop `shape: 'circle' | 'rounded-square'` (افتراضي circle)
- إضافة prop `vipBorder?: string` لإطار ملون حسب مستوى VIP
- المربع المدور: `rounded-xl` بدل `rounded-full`

### 3. إعادة كتابة `CustomerMobileView.tsx`
- استبدال `DataCard` + `SwipeableRow` بـ `CustomerListCard`
- إضافة زر تبديل عرض (قائمة/شبكة) في الأعلى
- عرض Grid: شبكة تلقائية (`grid-cols-2` على 393px، `grid-cols-3` على أعرض)
- **Infinite scroll**: بدل `ServerPagination`، استخدام `IntersectionObserver` لتحميل الصفحة التالية تلقائياً
- Fade-in: كل بطاقة تظهر بـ `animate-fade-in` مع `animation-delay` تصاعدي

### 4. إعادة كتابة `CustomerStatsBar.tsx` (موبايل)
شريط مدمج يجمع الإحصاء الرئيسي + chips فلترة:

```text
┌──────────────────────────────────────┐
│  👥 125 عميل                         │
│  [الكل] [نشط ✓] [VIP] [مدين] [شركات]│
└──────────────────────────────────────┘
```

- العدد الإجمالي كبير وبارز
- Chips قابلة للضغط تقوم بالفلترة مباشرة (`onClick` → `onTypeChange`, `onVipChange`, `onStatusChange`)
- كل chip يعرض العدد الخاص به (مثلاً "نشط 98")
- تمرير أفقي إذا لم تتسع

### 5. تحسين `CustomerGridCard.tsx`
- خلفية: `bg-gradient-to-b from-primary/5`
- Avatar: مربع مدور + إطار VIP ملون
- حالة النشاط: نقطة لونية + `opacity-60` لغير النشط
- شريط ائتمان: يظهر دائماً إذا كان هناك حد ائتمان (ليس فقط > 40%)
- ألوان الرصيد: `getBalanceColor` مع 3 درجات (أخضر/أصفر/أحمر)

### 6. تعديل `CustomersPage.tsx`
- **بحث يختفي**: `IntersectionObserver` أو scroll listener يخفي شريط البحث عند التمرير لأسفل
- **Infinite scroll**: state `allPages` يتراكم + `isFetchingNextPage` + observer على آخر عنصر
- **تبديل عرض موبايل**: `viewMode` يعمل على الموبايل أيضاً (حالياً قائمة فقط)
- **ترتيب**: dropdown يدعم الخيارات الأربعة على الموبايل

### 7. مكون `CustomerEmptyState.tsx` (جديد)

```text
┌────────────────────────────────────┐
│        👥 (أيقونة كبيرة)          │
│     ابدأ بإدارة عملائك            │
│                                    │
│  ① أضف عميلك الأول                │
│  ② أنشئ فاتورة أو عرض سعر        │
│  ③ تتبع المدفوعات والأرصدة       │
│                                    │
│  [+ إضافة عميل]  [📥 استيراد]     │
└────────────────────────────────────┘
```
